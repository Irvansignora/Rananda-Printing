ℹ️ Cara Kerja Tema

🖥️ Preview Lokal
Klik Aktifkan → tema langsung tersimpan ke data di browser ini → buka index.html dan tema langsung berubah.
🌐 Deploy ke Semua Device
Setelah puas pilih tema → klik Export JSON → timpa file data.json di server/GitHub → semua pengunjung dari device apapun langsung dapat tema yang sama.
💡 Kenapa ini lebih baik dari localStorage?
Tema tersimpan di data.json — file yang sama dibaca oleh semua pengunjung dari semua device. Tidak ada lagi masalah "tema berbeda di HP vs laptop". Cukup export satu file, semua orang dapat tampilan yang sama.
